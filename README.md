# 🔥 Ofertas Imperdíveis - Site de Afiliados

Um site simples e moderno criado com HTML, CSS e JavaScript para divulgação de produtos de afiliados.

---

## 🌐 Sobre o projeto

Este projeto tem como objetivo criar uma página de ofertas onde o usuário pode visualizar produtos e ser redirecionado para links de afiliado.

O foco é simplicidade, velocidade e facilidade de uso no celular e no computador.

---

## ⚡ Funcionalidades

✔ Página inicial com produtos em destaque  
✔ Página de listagem de produtos  
✔ Barra de busca funcional  
✔ Layout responsivo (celular e PC)  
✔ Botões de compra com links de afiliado  
✔ Interface moderna estilo loja online  

---

## 🛠️ Tecnologias usadas

- HTML5  
- CSS3  
- JavaScript puro  

---

## 📱 Como funciona

1. O usuário acessa o site  
2. Visualiza os produtos disponíveis  
3. Clica em “Comprar agora”  
4. É redirecionado para o site de afiliado  

---

## 💡 Objetivo

Este projeto foi criado para aprender desenvolvimento web e também para testar estratégias de marketing de afiliados na internet.

---

## 🚀 Possíveis melhorias futuras

- Carrinho de compras simulado  
- Sistema de favoritos  
- Página individual para cada produto  
- Integração com banco de dados  
- Painel administrativo  

---

## 📌 Observação

Este projeto não realiza vendas diretas. Todos os links são de afiliados de terceiros.
